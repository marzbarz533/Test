# Snake Game Website

A simple, interactive Snake game built with HTML5, CSS3, and JavaScript. Perfect for hosting on GitHub Pages!

## Features

- 🎮 Classic Snake gameplay
- 🏆 Score tracking with persistent high score
- ⏸️ Pause/Resume functionality
- 📱 Responsive design (works on desktop and mobile)
- 🎨 Beautiful gradient UI with glowing effects
- ⌨️ Keyboard controls (Arrow Keys or WASD)

## How to Play

1. Click "Start Game" or press Space to begin
2. Use **Arrow Keys** or **WASD** to control the snake
3. Eat the red food to grow and earn points
4. Don't hit the walls or yourself!
5. Press Space or click "Pause" to pause the game
6. Your high score is saved locally in your browser

## Game Controls

- **Arrow Keys** or **WASD** - Move snake
- **Space** - Start/Pause game
- **Start Button** - Begin or restart game
- **Pause Button** - Pause/Resume game

## Deployment to GitHub Pages

1. Create a new repository on GitHub (e.g., `my-snake-game`)
2. Extract the .zip file contents
3. Push the files to your repository
4. Go to Settings → Pages → Select "main" branch
5. Your game will be live at `https://yourusername.github.io/my-snake-game`

Or you can simply extract and open `index.html` in your browser to play locally!

## Technical Details

- **HTML5 Canvas** for game rendering
- **Vanilla JavaScript** - No frameworks or libraries required
- **LocalStorage** for high score persistence
- **CSS Grid** layout for responsive design

## Browser Compatibility

Works in all modern browsers:
- Chrome/Edge
- Firefox
- Safari
- Mobile browsers
